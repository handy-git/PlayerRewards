### 一、简介

> PlayerRewards是一款有点好用的玩家奖励插件  
> 腐竹们通过简单的指令创建激活码和礼物
> 玩家使用激活码来获取奖励

1. 支持mysql和本地sqlite存储
2. 创建指定激活码
3. 批量创建一次性激活码
4. 直接创建奖励物品
